# Pain Points

## Attendance
In Washington State in order to receive state funding you must prove attendance through state approved software. https://dcyf.wa.gov/services/early-learning-providers/electronic-attendance-system/approved-systems 
The free software is buggy, does not work well, and cannot run on all phones making it a pain to sign in and out of. To be approved you must be able to meet the below requirements


### 1. Technical Specifications (WAC 110-300-0455)
* **Authentication:** Must use PIN, biometric, or electronic signature.
* **Transactions:** Each check-in/out must create a unique, timestamped record.
* **Identity:** Must log the specific name of the authorized adult for every entry.
* **Retention:** 5-year secure, searchable database storage mandate.

### 2. Data Structure
* **Provider ID:** Must link records to the Social Services Payment System (SSPS) ID.
* **Export Format:** Must generate "Daily Attendance Detail" reports.
* **Audit Trail:** Must capture the "how" (e.g., "PIN 1234") and "who" for every event.

### 3. Approval Workflow
1.  Complete **DCYF Form 06-010**.
2.  Generate a sample monthly attendance report from your system.
3.  Submit both to **electronic.attendance@dcyf.wa.gov**.
4.  Receive "Approved Vendor" status prior to using for subsidy billing.
